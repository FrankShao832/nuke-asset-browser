# Nuke Asset Browser v2.0.0 — 展示视频脚本（2 分钟版）

> **总时长：** ~2 分钟  
> **场景数：** 8 个  
> **风格：** 快节奏、无废话、画面即内容  
> **观众：** 合成师 / 管线 TD / 技术总监  
> **配乐：** 节奏感电子乐（鼓点踩操作节奏）  

---

## 场景 0：开场（4 秒）

**画面：** 黑底

```
（居中）
Nuke Asset Browser  v2.0.0
```

**无文字标注。**

---

## 场景 1：打开浏览器（10 秒）

**画面：** Nuke 界面 → 按 `Ctrl+Shift+B` → 浏览器从右侧弹出

**操作：** 只这一个动作

**屏幕文字（fade-in，下 1/3）：**

```
Nuke 内部的共享便利店
合成师共用的模板工具库 · 一个界面 · 搜索即得
```

→ **0.3s 黑场**

---

## 场景 2：整体界面 + 搜索过滤（15 秒）

**画面：** 浏览器全屏，鼠标从顶栏扫到底栏 → 右侧边栏

**操作：**
1. 搜索框输入 `grain` → 网格实时过滤 → 清除恢复
2. 点击侧边栏 `Favorites` → `My Drafts`

**屏幕文字：**

```
PySide6 原生 · 自定义 FlowLayout · 6 卡片/行
实时搜索 300ms 防抖 · 侧边栏一键过滤
```

→ **0.3s 黑场**

---

## 场景 3：缩略图（12 秒）

**画面：** 依次划过 4 张卡片

**操作：** 鼠标依次悬停显示缩略图：EXR → DPX → MOV → Sequence

**屏幕文字（逐行出现，与操作同步）：**

```
EXR · OpenEXR + numpy · gamma 2.2
DPX · MOV · 序列 · ffmpeg 解码
5 级降级链路 · 永不空白
```

→ **0.3s 黑场**

---

## 场景 4：悬停播放 ⭐（15 秒）

**画面：** 聚焦序列卡片和视频卡片，反复移入/移出

**操作：**
1. 鼠标移入 Sequence → 24fps 帧轮播 → 移出 → 停止
2. 鼠标移入视频 → 同效果

**屏幕文字（两行依次出现）：**

```
悬停 → 24fps 帧轮播 · 移出即停
视频均匀抽帧 · 首播缓存 · 第二次秒开
```

→ **0.3s 黑场**

---

## 场景 5：拖拽导入 + 进度条（15 秒）

**画面：** Finder（上）→ 拖到浏览器（下）

**操作：**
1. 拖一个文件夹 → 进度条 "Processing…" → 新卡片出现
2. 批量拖入 5 个文件 → "Importing 3/5…"

**屏幕文字：**

```
拖文件夹 → 自动检测序列 · 合成一个草稿 · 不拆帧
批量拖入 → 进度条实时显示 · 导入完成 Toast 通知
```

→ **0.3s 黑场**

---

## 场景 6：Nuke 双向集成（15 秒）

**画面：** 分屏——Nuke Node Graph（左）+ 浏览器（右）

**操作（方向一）：**
1. 选中 Grade 节点 → 右键 `Save to Asset Browser`
2. 保存 → 浏览器出现新卡片

**操作（方向二）：**
3. 从浏览器拖一个 .nk 模板到 Node Graph → 节点出现

**屏幕文字：**

```
Node Graph 选中节点 → 右键保存到浏览器
浏览器拖出模板 → Nuke 原生导入
双向互操作 · 不离开 Nuke
```

→ **0.3s 黑场**

---
