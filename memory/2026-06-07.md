## EXR 缩略图与序列播放最终修复

### 问题
EXR 序列草稿缩略图显示异常（过暗/过亮）、悬停播放卡顿、缺帧跳跃

### 解决
1. **EXR 缩略图读取**：安装 OpenEXR + numpy，`_load_exr_thumbnail()` 解码 EXR → downsample ≤512px → gamma 2.2 → QImage → QPixmap
2. **色彩空间**：纯 gamma 2.2 校正，不做自动曝光或 Reinhard tonemap，**与 Nuke Viewer 行为一致**
3. **关键修复**：EXR 可能有负值像素，需要在 tonemap 前 `np.maximum(rgb, 0.0)` clamp，否则 `power(1/2.2)` 产生 NaN
4. **悬停播放性能**：首次播放时边播边缓存成 PNG（`play_{draft_id}_{frame_index:04d}.png`），后续秒切
5. **缺帧填补**：不存在的帧用上一帧填充，避免索引跳跃
